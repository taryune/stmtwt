stmtwt
====

Overview

Tweet your Steam Status.  

## Requirement
Python 3.6  
requests 2.14.2  
requests\_oauthlib 0.8.0  

## Usage
1. Edit config.py  
```
    # Twitter  
    CK = 'CONSUMER_KEY'  
    CS = 'CONSUMER_SECRET'  
    AT = 'ACCESS_TOKEN'  
    AS = 'ACCESS_TOKEN_SECRET'  

    # Steam API Key  
    SteamAPIKey = 'STEAMAPIKEY'  

    # Your SteamID64  
    steamid = 'STEAMID64'  
```
2. Run
    python stmtwt.py

3. Kill
    Press C-c
